# 运行与部署（AI Studio 应用）

这里包含了在本地运行该应用所需的一切。

在 AI Studio 查看应用：`https://ai.studio/apps/bundled/gemini_runner`

## 本地运行

**前置条件：** Node.js


1. 安装依赖：
   `npm install`
2. （可选）如果你计划添加 Gemini API 调用，可在 [.env.local](.env.local) 中设置 `GEMINI_API_KEY`
3. 启动开发服务器：
   `npm run dev`
